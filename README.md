###Pop组件
1，介绍：主要适用气泡效果类的弹层

2，固定属性：
           trigger: 触发方式，默认是hover；
           title: 气泡中的标题；
           content: 气泡主题内容；
           placement: 气泡显示位置；
3，demo：
    <Pop
        trigger="click"
        title="例子"
        content="主题内容"
        placement="left"
    >
      <button>click</button>
    </Pop>
